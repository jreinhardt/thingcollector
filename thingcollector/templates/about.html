About

The thingcollector is a web application that provides a full text search for the thingtracker network. It is written in python, using flask and whoosh, with easy deployment on OpenShift in mind. The frontend is based on bootstrap.

It is licensed under the AGPL3.

Contribute

You can help by publishing things on the thingtracker network, submit trackers to the collector, run your own collector, or hack the source.
